# 🧩 2884 알람 시계
- [문제 바로가기](https://www.acmicpc.net/problem/2884)
- **난이도** : 브론즈 3
- **알고리즘 분류** : 수학, 사칙연산
- **언어** : Java 11

## 🔎 문제 분석
**문제 요약**
- 현재 설정한 알람 시간보다 45분 앞서는 시간을 구하는 문제

**입력**
- 첫째 줄 : 공백으로 구분된 알람 시간 `H`시와 `M`분

**입력 조건**
- 0 <= `H` <= 23
- 0 <= `M` <= 59

**출력**
- 45분 앞서는 시간의 시 `H`와 분 `M`을 공백으로 구분해 출력

**제한 사항**
- 24시간제 : 시를 24 이상으로 표현 불가

## ♟️ 접근
조건문으로 분이 45분 이상일 때와 미만일 때를 나눠서 접근

### 💡 핵심 로직
- 분이 45분 **이상**일 때
  - 단순 뺄셈 후 출력

- 분이 45분 **미만**일 때
	- `H`에서 1을 빼고 `M`에 60을 가져온 뒤 계산
	- `H`가 음수가 되면 명시적으로 `23`으로 변경

### ✍🏻 구현
1. 공백으로 구분된 시 `H`와 분 `M` 입력
2. 조건문
	1. `M`이 45 미만일 때 : `H - 1`, `M + 15`
		1. `H`가 음수일 때 : `H = 23`
	2. `M`이 45 이상일 때 : `M - 45`
3. `H`와 `M`을 공백으로 구분해서 출력

## 👩🏻‍💻 코드
```java
import java.io.*;
import java.util.*;

public class Main {
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st = new StringTokenizer(br.readLine());
		
		int H = Integer.parseInt(st.nextToken());
		int M = Integer.parseInt(st.nextToken());
		
		if (M < 45) {
			H--;
			M += 15;
			
			if (H < 0) {
				H = 23;
			}
			System.out.println(H + " " + M);
		} else {
			M -= 45;
			System.out.println(H + " " + M);
		}
	}
}
```
