- **알고리즘 분류** : 수학, 기하학

## 🔎 문제 분석
**문제 요약**
- 삼각형 ABC의 각 BAC의 이등분선과 변 BC의 교점 M이 있다.
- 변 AB의 길이와 변 AC의 길이가 주어질 때 `(선분 BM의 길이) / (선분 CM의 길이)`의 값을 구하는 문제

**입력**
- 첫째 줄 : 공백으로 구분된 `c`와 `b`
	- `c` : 변 AB의 길이
	- `b` : 변 AC의 길이

**입력 조건**
- 1 <= `c`, `b` <= 1,000,000,000

**출력**
- `(선분 BM의 길이) / (선분 CM의 길이)`의 값

## ♟️ 접근
**각의 이등분선 정리**\
삼각형에서 어떤 각을 정확히 반으로 나누는 선인 각의 이등분선을 그으면, 마주보는 변을 나머지 두 변의 길이의 비로 내분\
삼각형 ABC에서 각 BAC의 이등분선의 비율 관계를 이용하는 문제

### 💡 핵심 로직
각 BAC에서 내려온 선이 각을 반으로 나누면 변 BC는 선분 BM과 선분 CM으로 나뉨
`BM : CM = AB : AC`이므로 구해야 하는 값은 `c / b`
- `c` : 변 AB의 길이
- `b` : 변 AC의 길이

### ✍🏻 구현
1. 공백으로 구분된 `c`와 `b` 입력
2. `c / b` 출력

## 👩🏻‍💻 코드
```java
import java.io.*;
import java.util.*;

public class Main {
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st = new StringTokenizer(br.readLine());
		
		double c = Double.parseDouble(st.nextToken());
		double b = Double.parseDouble(st.nextToken());
		
		System.out.println(c / b);
	}
}
```
