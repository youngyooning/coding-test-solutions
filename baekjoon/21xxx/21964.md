# 🧩 21964 선린인터넷고등학교 교가
- [문제 바로가기](https://www.acmicpc.net/problem/21964)
- **난이도** : 브론즈 4
- **알고리즘 분류** : 문자열
- **언어** : Java 11

## 🔎 문제 분석
**문제 요약**
- 문자열의 마지막 5글자를 출력하는 문제

**입력**
- 첫째 줄 : 문자열의 길이 `N`
- 둘째 줄 : `N`글자로 이루어진 공백 없는 문자열 `S`

**입력 조건**
- 5 <= `N` <= 100,000
- `S` : 알파벳 대소문자, 쉼표(`,`), 마침표(`.`)로만 구성되어 있음
	- 쉼표와 마침표도 한 글자로 간주

**출력**
- `S`의 마지막 5글자

## ♟️ 접근
`substring()`으로 문자열의 마지막 5글자만 추출\
문자열의 끝에서 5글자가 시작되는 위치인 `N - 5`가 시작 인덱스

### 💡 핵심 로직
`substring(beginIndex)`
- `S.substring(N - 5)` : 문자열 `S`의 `N-5`번째 인덱스부터 끝까지 반환

### ✍🏻 구현
1. `N` 입력
2. `S` 입력
3. `S.substring(N - 5)` 출력

## 👩🏻‍💻 코드
```java
import java.io.*;
  
public class Main {
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		
		int N = Integer.parseInt(br.readLine());
		String S = br.readLine();
		
		System.out.println(S.substring(N - 5));
	}
}
```

## ⏱️ 복잡도
**시간 복잡도** : O(1)
- 입력 크기가 커져도 항상 마지막 5글자만 다룸

**공간 복잡도** : O(1)
- 출력할 5글자만 다룸

## 👉🏻 복습 포인트
`substring()` : 문자열의 일부 범위를 추출하는 메서드
- `substring(beginIndex)` : beginIndex부터 문자열 끝까지 반환
- `substring(beginIndex, endIndex)` : beginIndex부터 endIndex 바로 직전까지 반환 (endIndex는 포함되지 않음)
