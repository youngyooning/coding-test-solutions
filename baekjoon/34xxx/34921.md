# 🧩 34921 덕후
- [문제 바로가기](https://www.acmicpc.net/problem/34921)
- **난이도** : 브론즈 5
- **알고리즘 분류** : 수학, 사칙연산
- **언어** : Java 11

## 🔎 문제 분석
**문제 요약**
- 주어진 수식에 따라 덕후력을 계산하는 문제

**입력**
- 첫째 줄 : 공백으로 구분된 나이 `A`와 solved.ac 티어 `T`

**입력 조건**
- 0 <= `A` <= 100
- 0 <= `T` <= 31

**출력**
- 덕후력 `P` 출력
	- `P`가 0보다 작으면 0 출력

## ♟️ 접근
결과값이 음수가 나오면 0을 출력해야 할 때 가능한 방식
1. if 문 또는 삼항 연산자
2. `Math.max(x, y)`

### 💡 핵심 로직
**Math.max(P, 0)**
- `P`가 0보다 작을 때 : `0` 반환
- `P`가 0 이상일 때 : `P` 반환

### ✍🏻 구현
1. 공백으로 구분된 나이 `A`와 solved.ac 티어 `T` 입력
2. `10 + 2 * (25 - A + T)` 수식 계산
3. `Math.max(P, 0)`로 덕후력 `P` 출력

## 👩🏻‍💻 코드
```java
import java.io.*;
import java.util.*;

public class Main {
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st = new StringTokenizer(br.readLine());
		
		int A = Integer.parseInt(st.nextToken());
		int T = Integer.parseInt(st.nextToken());
		
		int P = 10 + 2 * (25 - A + T);
		
		System.out.println(Math.max(P, 0));
	}
}
```

## ⏱️ 복잡도
**시간 복잡도** : O(1)
- 반복문이 없고 항상 "입력 → 사칙연산 → 출력"으로 고정 횟수의 연산 수행

**공간 복잡도** : O(1)
- 변수 `A`, `T`, `P`만 사용

## 👉🏻 복습 포인트
- `Math.max(x, y)` : x와 y 중 큰 값 반환
- `Math.min(x, y)` : x와 y 중 작은 값 반환
