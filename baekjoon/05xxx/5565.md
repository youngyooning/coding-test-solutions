# 🧩 5565 영수증
- [문제 바로가기](https://www.acmicpc.net/problem/5565)
- **난이도** : 브론즈 4
- **알고리즘 분류** : 수학, 구현, 사칙연산
- **언어** : Java 11

## 🔎 문제 분석
**문제 요약**
- 책 10권의 총 가격과 책 9권의 가격을 알고 있을 때, 나머지 책 1권의 가격을 구하는 문제

**입력**
- 첫째 줄 : 책 10권의 총 가격
- 둘째 줄 ~ 아홉째 줄 : 책 9권의 가격

**입력 조건**
- 책의 가격 : 10,000 이하인 양의 정수

**출력**
- 가격을 읽을 수 없는 책의 가격

## ♟️ 접근
총합 - 알고 있는 값들의 누적 합

### ✍🏻 구현
1. 책 10권의 총 가격 `total` 입력
2. 책 9권의 총 가격을 저장할 변수 `sum` 생성
3. for문으로 9번 반복
	1. 한 줄씩 읽으며 `sum`에 누적 합 저장
4. `total - sum` 출력

## 👩🏻‍💻 코드
```java
import java.io.*;
  
public class Main {
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		
		int total = Integer.parseInt(br.readLine());
		int sum = 0;
		
		for (int i = 0; i < 9; i++) {
			sum += Integer.parseInt(br.readLine());
		}
		
		System.out.println(total - sum);
	}
}
```

- 별도의 자료구조에 저장 없이 즉시 누적 합을 구함
