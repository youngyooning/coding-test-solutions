# 🧩 32951 AI 선도대학
- [문제 바로가기](https://www.acmicpc.net/problem/32951)
- **난이도** : 브론즈 5
- **알고리즘 분류** : 수학, 사칙연산
- **언어** : Java 11

## 🔎 문제 분석
**문제 요약**
- 특정 연도가 2024년으로부터 몇 년이 지났는지 구하는 문제

**입력**
- 첫째 줄 : 연도 `N`

**입력 조건**
- 2024 <= `N` <= 9999

**출력**
- `N - 2024`

## ✍🏻 구현
1. `N` 입력
2. `N - 2024` 출력

## 👩🏻‍💻 코드
```java
import java.io.*;

public class Main {
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		
		int N = Integer.parseInt(br.readLine());
		
		System.out.println(N - 2024);
	}
}
```

## ⏱️ 복잡도
**시간 복잡도** : O(1)
- 입력 크기가 커져도 항상 "입력 → 뺄셈 연산 → 출력"으로 고정 횟수의 연산 수행

**공간 복잡도** : O(1)
- 입력 크기와 관계없이 항상 변수 `N` 사용
