# 🧩 31822 재수강
- [문제 바로가기](https://www.acmicpc.net/problem/31822)
- **난이도** : 브론즈 4
- **알고리즘 분류** : 구현, 문자열
- **언어** : Java 11

## 🔎 문제 분석
**문제 요약**
- 과목 코드의 앞 5자리가 일치하는 과목의 개수를 구하는 문제

**입력**
- 첫째 줄 : 재수강할 과목의 과목 코드
- 둘째 줄 : 수강 신청 가능한 과목의 개수 `N`
- 셋째 줄 ~ `N`번째 줄 : 수강 신청 가능한 과목의 서로 다른 과목 코드

**입력 조건**
- 1 <= `N` <= 16

**출력**
- 재수강으로 인정되는 과목의 개수 = 과목 코드의 앞 5자리가 일치하는 과목의 개수

## ♟️ 접근
문자열의 앞 5자리를 부분 비교하는 문제\
문자열의 특정 prefix 비교 문제에서 가능한 방법
- `substring()` 비교
- `startsWith()`

### 💡 핵심 로직
`str.substring(0, 5).equals(code.substring(0, 5))`
- `substring()` 메서드로 과목 코드의 앞 5자리 추출
- `equals()` 메서드로 추출한 두 문자열이 같은지 비교

### ✍🏻 구현
1. 재수강할 과목의 과목 코드 `code` 입력
2. 수강 신청 가능한 과목의 개수 `N` 입력
3. 과목 코드의 앞 5자리가 일치하는 과목의 개수를 저장할 변수 `count` 생성
4. for문으로 `N`번 반복
	1. 수강 신청 가능한 과목의 과목 코드 `str` 입력
	2. if문으로 과목 코드의 앞 5자리가 같은지 비교
	3. 같으면 `count` 1 증가
5. `count` 출력

## 👩🏻‍💻 코드
```java
import java.io.*;

public class Main {
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		
		String code = br.readLine();
		int N = Integer.parseInt(br.readLine());
		int count = 0;
		
		for (int i = 0; i < N; i++) {
			String str = br.readLine();
			if (str.substring(0, 5).equals(code.substring(0, 5))) {
				count++;
			}
		}
		
		System.out.println(count);
	}
}
```

## ⏱️ 복잡도
**시간 복잡도** : O(N)
- `N`번 반복

## 🍯 해결 팁
prefix 비교 문제에서는 보통 `startsWith()` 메서드 사용
- `substring()` : 새로운 문자열 객체를 생성해 비교 횟수가 많을 경우 성능 저하 발생 가능
- `startsWith()` : 내부적으로 문자 배열을 직접 비교해 추가 객체 생성이 없음

`N`이 최대 16이므로 `substring()` 메서드를 사용했다.

## 👉🏻 복습 포인트
`substring()` : 문자열의 일부 범위를 추출하는 메서드
- `substring(beginIndex)` : beginIndex부터 문자열 끝까지 반환
- `substring(beginIndex, endIndex)` : beginIndex부터 endIndex 바로 직전까지 반환 (endIndex는 포함되지 않음)
