# 🧩 1546 평균
- [문제 바로가기](https://www.acmicpc.net/problem/1546)
- **난이도** : 브론즈 1
- **알고리즘 분류** : 수학, 사칙연산
- **언어** : Java 11

## 🔎 문제 분석
**문제 요약**
- 점수들을 최댓값 기준으로 조작한 뒤 새로운 평균을 구하는 문제

**입력**
- 첫째 줄 : 과목(점수)의 개수 `N`
- 둘째 줄 : 공백으로 구분된 `N`개의 점수

**입력 조건**
- `N` <= 1000
- 점수 : 0 이상 100 이하의 정수로, 적어도 하나의 값은 0보다 큼

**출력**
- 조작된 점수들의 평균

## ♟️ 접근
조작된 점수와 평균은 소수가 나올 수 있으므로 `double` 타입 또는 실수 리터럴 사용\
`50 / 70 * 100`을 정수 연산으로 하면 `50 / 70 = 0`이 되어버려서 틀린 값이 나옴

### 💡 핵심 로직
1. 점수를 조작하려면 최댓값이 필요하므로 먼저 배열에 점수를 저장하면서 최댓값을 찾음
2. `(점수 / 최댓값) * 100` 공식을 반복문으로 모든 점수에 적용하면서 누적 합을 계산

### ✍🏻 구현
1. 과목(점수)의 개수 `N` 입력
2. 배열에 점수를 저장하면서 순회하며 최댓값 찾기
3. 다시 순회하면서 `(점수 / 최댓값) * 100`을 누적 합에 더함
4. 누적 합의 평균 출력

## 👩🏻‍💻 코드
```java
import java.io.*;
import java.util.*;

public class Main {
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		int N = Integer.parseInt(br.readLine());
		
		int[] score = new int[N];
		int M = 0;
		double sum = 0;
		
		StringTokenizer st = new StringTokenizer(br.readLine());
		for (int i = 0; i < N; i++) {
			score[i] = Integer.parseInt(st.nextToken());
			if (score[i] > M) {
				M = score[i];
			}
		}
		for (int i = 0; i < N; i++) {
			sum += (double) score[i] / M * 100;
		}
		
		System.out.println(sum / N);
	}
}
```

## ⏱️ 복잡도
**시간 복잡도** : O(N)
- 입력된 점수를 입력 크기만큼 순회

**공간 복잡도** : O(N)
- 입력된 점수를 입력 크기만큼의 배열에 저장

## 🍯 해결 팁
- 평균처럼 연산 결과가 소수로 나올 수 있을 때는 `double` 타입 또는 실수 리터럴 사용

## 👉🏻 복습 포인트
정수 나눗셈 vs 실수 나눗셈
- `50 / 70 = 0` : 정수 나눗셈
- `50.0 / 70.0 = 0.7142857142857143` : 실수 나눗셈
