# 🧩 27323 직사각형
- [문제 바로가기](https://www.acmicpc.net/problem/27323)
- **난이도** : 브론즈 5
- **알고리즘 분류** : 수학, 기하학, 사칙연산
- **언어** : Java 11

## 🔎 문제 분석
**문제 요약**
- 직사각형의 세로와 가로 길이가 주어졌을 때, 직사각형의 넓이를 구하는 문제

**입력**
- 첫째 줄 : 세로 길이 `A`
- 둘째 줄 : 가로 길이 `B`

**입력 조건**
- 1 <= `A`, `B` <= 100
- `A`, `B` : 정수

**출력**
- `A`와 `B`의 곱

## ✍🏻 구현
1. 세로 길이 `A` 입력
2. 가로 길이 `B` 입력
3. `A * B` 출력

## 👩🏻‍💻 코드
```java
import java.io.*;

public class Main {
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		
		int A = Integer.parseInt(br.readLine());
		int B = Integer.parseInt(br.readLine());
		
		System.out.println(A * B);
	}
}
```

## ⏱️ 복잡도
**시간 복잡도** : O(1)
- 입력 크기가 커져도 항상 "입력 → 곱셈 연산 → 출력"으로 고정 횟수의 연산 수행

**공간 복잡도** : O(1)
- 입력 크기와 관계없이 항상 변수 `A`, `B` 사용
