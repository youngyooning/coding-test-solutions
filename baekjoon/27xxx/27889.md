# 🧩 27889 특별한 학교 이름
- [문제 바로가기](https://www.acmicpc.net/problem/27889)
- **난이도** : 브론즈 5
- **알고리즘 분류** : 구현, 문자열
- **언어** : Java 11

## 🔎 문제 분석
**문제 요약**
- 학교의 약칭이 주어졌을 때 정식 명칭을 구하는 문제

**입력**
- 첫째 줄 : 학교의 약칭

**출력**
- 입력된 학교의 정식 명칭

## ♟️ 접근
### 💡 핵심 로직
**switch 문**\
switch 문의 조건식의 결과와 일치하는 case 문이 있다면 해당 case 문에 속한 문장이 실행됨

### ✍🏻 구현
1. 학교의 약칭 `school` 입력
2. switch 문
	1. case 문으로 학교의 약칭 `school`과 일치하는 정식 명칭 `full` 작성
	2. break 문
3. `full` 출력

## 👩🏻‍💻 코드
```java
import java.io.*;
  
public class Main {
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		
		String school = br.readLine();
		String full;
		
		switch(school) {
			case "NLCS":
				full = "North London Collegiate School";
				break;
				
			case "BHA":
				full = "Branksome Hall Asia";
				break;
				
			case "KIS":
				full = "Korea International School";
				break;
				
			default:
				full = "St. Johnsbury Academy";
		}
			
		System.out.println(full);
	}
}
```

## ⏱️ 복잡도
**시간 복잡도** : O(1)
- 반복문이 없고 항상 "입력 → switch 문 → 출력" 연산을 수행하므로 상수 시간

**공간 복잡도** : O(1)
- 추가로 사용하는 메모리가 없으므로 상수 공간

## 🍯 해결 팁
- 입력값이 고정되었을 때는 if 문보다 switch 문의 가독성이 좋음

## 👉🏻 복습 포인트
- break 문 : case 문 실행 후 다음 case 문으로 진행되는 것을 막고 switch 문을 빠져나가게 함
