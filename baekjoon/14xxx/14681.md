# 🧩 14681 사분면 고르기
- [문제 바로가기](https://www.acmicpc.net/problem/14681)
- **난이도** : 브론즈 5
- **알고리즘 분류** : 구현, 기하학
- **언어** : Java 11

## 🔎 문제 분석
**문제 요약**
- 점의 좌표 (x, y)가 주어질 때, 해당 점이 어느 사분면에 속하는지 구하는 문제

**입력**
- 첫째 줄 : `x`
- 둘째 줄 : `y`

**입력 조건**
- -1000 <= `x`, `y` <= 1000
- `x`, `y` != 0

**출력**
- 점 (x, y)가 속한 사분면 번호 1, 2, 3, 4 중 하나

## ♟️ 접근
if...else문으로 조건 분기

### ✍🏻 구현
1. `x` 입력
2. `y` 입력
3. 조건 분기
	1. 1사분면 (x > 0 && y > 0)
	2. 2사분면 (x < 0 && y > 0)
	3. 3사분면 (x < 0 && y < 0)
	4. 4사분면 (x > 0 && y < 0)

## 👩🏻‍💻 코드
```java
import java.io.*;

public class Main {
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		
		int x = Integer.parseInt(br.readLine());
		int y = Integer.parseInt(br.readLine());
		
		if (x > 0 && y > 0) {
			System.out.println(1);
		} else if (x < 0 && y > 0) {
			System.out.println(2);
		} else if (x < 0 && y < 0) {
			System.out.println(3);
		} else {
			System.out.println(4);
		}
	}
}
```

**공간 복잡도** : O(1)
- 입력 크기와 관계없이 항상 변수 `x`, `y` 사용
