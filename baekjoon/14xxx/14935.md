# 🧩 14935 FA
- [문제 바로가기](https://www.acmicpc.net/problem/14935)
- **난이도** : 브론즈 4
- **알고리즘 분류** : 수학, 구현, 애드 혹
- **언어** : Java 11

## 🔎 문제 분석
**문제 요약**\
숫자 `x`가 주어졌을 때 해당 숫자가 FA 수인지 판별하는 문제
- FA 수 : 첫 자리 숫자와 자리수를 곱한 결과를 반환하는 함수를 반복해서 수행했을 때 어느 시점에서부터 동일한 수가 나오는 수

**입력**
- 첫째 줄 : `x`

**입력 조건**
- 0 <= `x` <= 10<sup>100</sup>

**출력**
- `x`가 FA 수일 때 : `FA`
- `x`가 FA 수가 아닐 때 : `NFA`

## ♟️ 접근
수학적 관찰이 필요한 문제
- `F(x)` = 첫 자리 숫자 * 자리수

모든 자연수는 `F(x)`를 반복 적용하면 값은 계속해서 작아지고 결국 한 자리 수가 됨\
한 번 한 자리 숫자가 되면 해당 숫자는 자기 자신으로 고정되므로 모든 자연수는 FA 수

## 👩🏻‍💻 코드
```java
public class Main {
	public static void main(String[] args) {
		System.out.println("FA");
	}
}
```
