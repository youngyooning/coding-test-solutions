# 🧩 10811 바구니 뒤집기
- [문제 바로가기](https://www.acmicpc.net/problem/10811)
- **난이도** : 브론즈 2
- **알고리즘 분류** : 구현, 시뮬레이션
- **언어** : Java 11

## 🔎 문제 분석
**문제 요약**
- 1번부터 `N`번까지 번호가 순서대로 적혀 있는 `N`개의 바구니가 있을 때, 특정 구간의 순서를 역순으로 `M`번 바꾼 뒤 바구니 번호의 최종 순서를 구하는 문제

**입력**
- 첫째 줄 : 공백으로 구분된 바구니 개수 `N`과 순서를 역순으로 바꿀 횟수 `M`
- 둘째 줄 ~ `M`번째 줄 : 공백으로 구분된 바구니의 순서를 역순으로 만드는 구간인 `i`와 `j`
	- `i` : 시작점 (`i`번 바구니부터)
	- `j` : 끝점 (`j`번 바구니까지)

**입력 조건**
- 1 <= `N`, `M` <= 100
- 1 <= `i` <= `j` <= `N`

**출력**
- 바구니의 특정 구간의 순서를 역순으로 바꾸는 연산을 `M`번 수행한 뒤, 바구니 번호의 최종 순서를 왼쪽부터 공백으로 구분해 출력

## ♟️ 접근
배열의 특정 구간을 역순으로 바꾸는 것을 반복하는 구간 뒤집기 문제

가능한 방법
- 특정 구간을 임시 배열에 뒤집어서 저장 후 원본 배열에 복사
- 투 포인터 알고리즘

### 💡 핵심 로직
**투 포인터 알고리즘**\
시작과 끝을 가리키는 두 개의 포인터를 사용해 양 끝에서부터 중앙으로 이동하며 원소를 서로 교환
1. 시작점 `i`가 끝점 `j`보다 작을 때까지, 즉 중앙에서 만날 때까지 반복
	1. 역순으로 만들기 위해 `basket[i]`와 `basket[j]` 교환
	2. `i`는 1 증가, `j`는 1 감소하며 구간을 양쪽에서 좁혀가며 중앙으로 이동
2. `i`가 `j`보다 커지면 종료

### ✍🏻 구현
1. 바구니 개수 `N`과 뒤집기 횟수 `M` 입력
2. `N + 1` 크기의 배열 생성 후 1부터 `N`까지 초기화
3. 반복문으로 구간 뒤집기를 `M`번 수행
	1. 뒤집을 구간의 시작점 `i`와 끝점 `j` 입력
	2. `i` ~ `j` 구간 뒤집기
4. `StringBuilder`와 `append()` 메서드로 배열 값과 공백을 이어 붙여서 출력

## 👩🏻‍💻 코드
```java
import java.io.*;
import java.util.*;
  
public class P10811 {
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st = new StringTokenizer(br.readLine());
		
		int N = Integer.parseInt(st.nextToken());
		int M = Integer.parseInt(st.nextToken());
		int[] basket = new int[N + 1];
		
		for (int i = 1; i <= N; i++) {
			basket[i] = i;
		}
		
		for (int k = 0; k < M; k++) {
			st = new StringTokenizer(br.readLine());
			int i = Integer.parseInt(st.nextToken());
			int j = Integer.parseInt(st.nextToken());
			
			while (i < j) {
				int tmp = basket[i];
				basket[i] = basket[j];
				basket[j] = tmp;
				i++;
				j--;
			}
		}
		
		StringBuilder sb = new StringBuilder();
		for (int i = 1; i <= N; i++) {
			sb.append(basket[i]).append(' ');
		}
		
		System.out.println(sb);
	}
}
```

## ⏱️ 복잡도
**시간 복잡도** : O(NM)
- 입력이 `N`과 `M` 두 개 존재
- 바깥 for문을 `M`번 반복하는데, 그 안쪽 while문으로 뒤집기 최대 `N`번 가능

**공간 복잡도** : O(N)
- 바구니 번호 `N`개를 배열에 저장

## 🍯 해결 팁
- 바구니 번호가 1부터 시작하므로 인덱스를 헷갈리지 않도록 `N + 1` 크기의 배열을 만들어 `basket[1]`부터 시작하면 실수를 줄일 수 있음
- 데이터가 정렬되어 있거나 연속적인 특징이 있을 때, 전체 또는 특정 구간을 뒤집을 때 투 포인터 알고리즘 사용

## 👉🏻 복습 포인트
- `StringBuilder` : 문자열을 효율적으로 다룰 수 있게 하는 클래스로, 문자열을 계속 이어 붙여서 한 번에 빠르게 출력 가능
- `append()` : 문자열 뒤에 새로운 문자열을 이어 붙이는 메서드로, `StringBuilder`에서 자주 사용
