# 🧩 10699 오늘 날짜
- [문제 바로가기](https://www.acmicpc.net/problem/10699)
- **난이도** : 브론즈 5
- **알고리즘 분류** : 구현
- **언어** : Java 11

## 🔎 문제 분석
**문제 요약**
- 서울의 오늘 날짜를 `YYYY-MM-DD` 형식으로 출력하는 문제

**출력**
- 서울의 오늘 날짜를 `YYYY-MM-DD` 형식으로 출력

## ♟️ 접근
제출 시점의 서울 날짜를 출력해야 함\
Java의 날짜 API `LocalDate`를 사용해서 현재 서울의 날짜 구하기

### 💡 핵심 로직
`LocalDate.now(ZoneId.of("Asia/Seoul"))`

### ✍🏻 구현
1. 서울의 오늘 날짜 구하기
2. 날짜 출력

## 👩🏻‍💻 코드
```java
import java.time.*;

public class Main {
	public static void main(String[] args) {
		LocalDate today = LocalDate.now(ZoneId.of("Asia/Seoul"));
		System.out.println(today);
	}
}
```

## ⏱️ 복잡도
**시간 복잡도** : O(1)
- 현재 날짜를 가져와 출력 한 번 수행

**공간 복잡도** : O(1)
- `LocalDate` 객체 하나 사용

## 🍯 해결 팁
- `LocalDate.now()`만 써도 정답 처리가 되지만 채점 서버의 시간대가 UTC이기 때문에 `ZoneId`를 서울로 지정하는 게 안전함

## 👉🏻 복습 포인트
- 날짜만 필요할 때 : `LocalDate`
- 날짜와 시간이 필요할 때 : `LocalDateTime`
- 날짜, 시간, 시간대가 필요할 때 : `ZonedDateTime`
